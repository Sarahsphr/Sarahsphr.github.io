Research Projects

Global Prevalence of Hepatitis B Virus Resistance Mutations

University of Chester | 2026

Developed a bioinformatics pipeline for large-scale analysis of more than 17,000 HBV genomes to identify clinically relevant antiviral resistance mutations. The project involved sequence filtering, genotyping, multiple sequence alignment, mutation screening, prevalence analysis, and visualization using Python, Biopython, MAFFT, and R.

Technologies: Python • Biopython • MAFFT • R

scRNA-seq Analysis of Craniosynostosis

Conducted single-cell RNA sequencing analysis to investigate cellular heterogeneity and transcriptional changes associated with craniosynostosis. The workflow included quality control, normalisation, clustering, dimensionality reduction, differential expression analysis, and cell-type annotation using Seurat in R.

scRNA-seq Analysis of Human Diabetic Kidney Disease

Conducted single-cell RNA sequencing analysis of human diabetic kidney disease datasets, including quality control, normalisation, clustering, UMAP visualisation, marker identification, pathway enrichment, and cell-cell communication analysis.

Whole Exome Sequencing Analysis of Drug-Resistant TNBC

Processed whole-exome sequencing data through quality control, alignment, BAM processing, variant calling, filtering, annotation, prioritisation, and pathway analysis using Samtools, GATK, ANNOVAR, SnpEff, IGV, and Cytoscape.

RNA-seq Transcriptomic Analysis

Performed end-to-end RNA-seq analysis, including quality control, alignment, differential expression analysis, and interpretation of transcriptional changes using DESeq2 and edgeR.

TCGA Data Retrieval and Analysis

Retrieved and processed RNA-seq datasets using TCGAbiolinks. Performed normalisation, batch correction, statistical analysis, cohort comparison, and visualisation using heatmaps and expression profiles.

Molecular Docking for Drug-Target Interaction Analysis

Applied molecular docking using AutoDock Vina to investigate binding interactions between candidate compounds and disease-related protein targets. Analysed binding energies and visualised docking results using PyMOL.

QA Automation with Python

Developed a Python-based workflow to automate pharmaceutical document processing and extract structured information from quality documents, supporting more efficient quality assurance review.
